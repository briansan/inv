    Logging In

    inv is currently only supported within Villanova University. To log in, use your Villanova User ID. (If you're seeing this, you probably already figured this part out...)

    Navigation

    inv uses BreadInterface, so learning how navigate through the app shouldn't take too long. In each menu, all the navigation buttons are located on the corners of the screen. Sometimes, the bottom-middle text is clickable too.

    Help menu: Clicking on the title of the menu (or the top-middle text will display a help menu describing the position, icon, and action of each button.

    Requirements

    The whole point of this software is to keep track of your belongings. So before you begin, make sure that you have some items that you want to catalog

    Barcode Scanner

    Using this software is made infinitely easier with the integration of a barcode scanner. You can access this feature by clicking the bottom-left button of the home menu. Hold it to the barcode of the asset's unique tag (EE) and the app will find the matching asset or display a menu to add that asset into the database if it doesn't exist. From there, you will want to click the add button on the bottom-right to add an inventory record of that asset. At that point, you have updated the system on that asset's whereabouts.

    Labels

    The first thing you want to do is to define your labels. The Item Category, Item Manufacturer, and Location Buildings are the three labels that you want to specify. For example, "Computer" would be an item category, "Dell" would be an item manufacturer, and "CEER" would be a location building.

    To do this, go to the settings menu where you'll see one button for each label type and clicking it will display a list of that specific label. At the bottom-right, there will be a plus button for adding labels.

    Note: you must have Operator level permissions to add, delete, or edit labels.

    Subentities

    Once you have defined your labels, now you can define your subentities: the Item and Location.
